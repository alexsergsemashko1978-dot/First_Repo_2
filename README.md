# Мой первый проект на Python

Это учебный репозиторий, созданный в процессе изучения основ программирования и работы с Git.

## Описание
В этом проекте я практикуюсь:
* Работать со словарями и множествами (`dict_and_set`).
* Настраивать интерпретатор в PyCharm.
* Использовать систему контроля версий Git.

## Как запустить
1. Установите Python 3.13.
2. Запустите файл `main.py`.
